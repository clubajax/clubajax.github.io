Club AJAX Libraries

Club AJAX contains several libraries that can be used for creating web applications. These libraries are combined into the metl project.

  • metl - Universal UI for web applications
  • mouse - Tracks mouse position for dragging objects
  • Date Picker - Calendar custom-element
  • create-element - Utility for easily creating and managing web components.
  • on - A simple library for handling DOM node events.
  • dom - A convenience library for common DOM methods.
  • dates - Small utility library for handling dates.
  • store - Data store for searching, sorting, and paginating items.
  • key-nav - Turn a list of items into keyboard navigable.
  • fx - Utility for node animations.
  • popup - Utility for opening elements as a popup.
